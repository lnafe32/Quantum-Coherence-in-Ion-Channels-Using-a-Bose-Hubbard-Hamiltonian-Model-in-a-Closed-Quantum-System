# Quantum Coherence in Ion Channels

This project explores the fascinating intersection of quantum computing and neuroscience. It investigates whether ion channels, the trans-membrane proteins responsible for neural signaling, might exploit quantum coherence to achieve the near-perfect ion transport efficiency observed in biological systems, despite the "warm, wet, and noisy" environment of the brain.

## Bose-Hubbard Phase Transition
Explore how changing the ratio of repulsion (U) to hopping (J) shifts the system between superfluidity and an insulating state.

## Project Structure
- `Quantum_Coherence.ipynb`: The primary Jupyter Notebook containing the formal simulation, PennyLane implementation, and mathematical discussion.
- `Bose_Hubbard_Visualizer.html`: Interactive web-based visualization of the Hamiltonian phase transition.
- `Ion_Channel_Quantum_Visualizer.html`: Interactive web-based simulation of ion transport dynamics.
